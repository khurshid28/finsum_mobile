# FinSum - Installation Guide

## Quick Start

### 1. Install Flutter
Make sure you have Flutter SDK installed (>=3.2.0)

### 2. Download Manrope Font
1. Go to: https://fonts.google.com/specimen/Manrope
2. Download all weights: Regular, Medium, SemiBold, Bold, ExtraBold
3. Create folder: `assets/fonts/`
4. Add these files:
   - `Manrope-Regular.ttf`
   - `Manrope-Medium.ttf`
   - `Manrope-SemiBold.ttf`
   - `Manrope-Bold.ttf`
   - `Manrope-ExtraBold.ttf`

### 3. Create Assets Folders
```bash
mkdir -p assets/images assets/icons assets/fonts
```

### 4. Install Dependencies
```bash
flutter pub get
```

### 5. Run the App
```bash
flutter run
```

## Test Credentials

### Login
- Phone: Any 9-digit number (e.g., 901234567)
- Format: +998 (XX) XXX-XX-XX

### Verify
- Code: Any 6-digit code (e.g., 123456)

### Passport
- Series: Any 2 uppercase letters (e.g., AB)
- Number: Any 7 digits (e.g., 1234567)
- Date: DD.MM.YYYY format (e.g., 20.10.1998)
